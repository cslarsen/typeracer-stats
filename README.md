TypeRacer stats in R
====================

Shows some statistic on your TypeRacer data.

Prerequisites
=============

You need the statistical package R.

How to use
==========

Download your CSV file from TypeRacer, then

    $ Rscript trstats.r race_data.zip

There are also options.
